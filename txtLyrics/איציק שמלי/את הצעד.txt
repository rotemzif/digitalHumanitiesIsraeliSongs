בחלום שלי את מתגשמת 
ובבוקר לי את הזריחה
את נוגעת בי רק את יודעת
את היחידה שמצליחה
את הקלף שלי שמנצח
מכירה אותי מקיר לקיר
לעירך נתת את המפתח
בעירי שלי את ראש העיר
את הצעד, את הדרך
את הבית, את השיר
את הרוח והשמש
בשמים שלי את האוויר
את הכוח את הרגש
את הטוהר והתום
את היד שמנחמת
ועוטפת את חיי בחום
בחיוך שלי את העניים
ובאביב את לי הצבעים
ובסוף כל יום את לי המים
שמרווים את צימאוני מזה שנים
